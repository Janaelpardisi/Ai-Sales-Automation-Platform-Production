# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-10

### Added
- Initial release of AI Sales Agent
- Multi-agent AI system with Research, Qualification, and Personalization agents
- Campaign management system with CRUD operations
- Email automation with SMTP integration (Gmail support)
- Lead discovery and qualification workflow
- AI-powered email personalization using Google Gemini
- Modern dashboard UI with Tailwind CSS
- RESTful API with FastAPI
- SQLite database with SQLAlchemy ORM
- Web scraping capabilities with Playwright
- Email templates system (initial, follow-up)
- Unsubscribe management
- Campaign analytics and statistics
- Comprehensive documentation (README, setup guides)
- Docker support with docker-compose
- MIT License

### Features
